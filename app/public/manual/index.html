180 Shooter

HALA Connected Shooter 180 App to track stats for basketball coaches and players

Getting Started

You should start by signing up for an account, if you intend to use sync/save features.

Click Login on the home screen and then click Create Account on the login screen. Enter your details and click submit. You will then be taken back to the login screen and you should login with the user/pass you entered for your account.

Sync Workouts

Overview

Sync workouts provides the facility for coaches to create workouts and send them to players. The players then complete the workouts and return them at which point the coach can view progress. Quick bit of terminolog: A requested workout is called a Routine. A practice is the players attempt at the workout, and has associated outcomes.

There are some prerequisites to be able to perform sync:

  • Both coach and player must be registered
  • The player must have email information in the team list
  • Connection to the internet is required to perform the actual sync

The sync workflow proceeds as follows:

  1. Coach creates a workout and tags it to be sent to player(s)
  2. Coach sends workouts to players
  3. Player(s) retrieve the workouts requested
  4. Player completes the request workouts
  5. Player sends completed workouts to coach
  6. Coach retrieves completed workouts

Creating Workouts

Initially the workouts list (accessed from home page -> workouts) will be empty. Click the plus (+) button in the top right to create a new workout and enter a name and description. Workouts comprise a list of drills - click the plus (+) next to the title Drills, above the drill table list. Select the drills you want to make up the workout and click Done. Then back in the workout create view click Save. Your workout list should now look like the image below:

Clicking the envelope looking icon* with the red arrow above it nagigates to the player selection view. Select the players you wish to send the workout request to.

*Must be registered

Perform the sync

Navigate to the sync view from the home page. The app will check the connection and requested workouts. See the image below for the buttons used for sync operations

The red buttons perform coach operations and the green buttons perform player operations.

Send Requests

The coach clicks the send requests button to send the request workouts to the targeted players

Retrieve Requests

A player clicks the retrieve requests button to fetch any workouts requested by coach. The retrieved requests will then appear in the Routine section of the workouts table list. Clicking on the basketball icon will create a practice allowing the player to fulfil the request - i.e. complete each drill in the workout and optionally complete outcome stats

Send Completed

A player clicks the send complete button after completing workouts requested by coach. This returns the workouts to the coach for inspection

Retrieve Completed

The coach clicks the retrieve completed button to check if there are any completed routines. The routines will appear in the practice report view, accessible from the home page -> Practice Report. Click on individial routines to see the players outcomes for the practice

Workout List Option

  • Clicking the new workout row will take you back to the workout detail view where you can edit your workout.
  • Clicking the red cross (X) will delete the workout.
  • Clicking the basket ball icon at the right of the row will navigate (a player) to views responsible for completing each of the drills that make up the workout.
